Resume Skills Comparison

This is a Streamlit application that processes resumes to compare skills and generate reports.

Features:
- Upload multiple PDF resumes
- Enter required skills for comparison
- View skills comparison in a bar graph
- Save results to a Word document
- Footer with creator information

Requirements:
- Streamlit
- PyMuPDF
- Transformers
- Langdetect
- Pandas
- Matplotlib
- python-docx

How to Run:
1. Install the required packages using `pip install -r requirements.txt`.
2. Run the Streamlit app with `streamlit run your_app_file.py`.
